Emby may use path configuration. Set these when running the container:

| Path                    | Description                           |
| ----------------------- | ------------------------------------- |
| `/path/to/tvshows`      | Path for TV shows in Emby.             |
| `/mnt/share1`           | Mount point for Emby to access data.   |
