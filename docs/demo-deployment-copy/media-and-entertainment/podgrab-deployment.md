### Path Configuration

"Podgrab" may use path configuration. Set these when running the container:

| Path                          | Description                         |
| ----------------------------- | ----------------------------------- |
| `/host/path/to/assets`        | Path for Podgrab assets.            |
| `/assets`                     | Internal path used by Podgrab.      |
